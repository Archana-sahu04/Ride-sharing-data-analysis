# Ride-sharing-data-analysis
This project analyzes ride-sharing data to understand ride demand patterns, weather impact, and time-based trends. The goal is to extract meaningful insights that can help ride-sharing companies improve service efficiency, allocate drivers effectively, and understand customer demand.

**Objectives**
- Identify peak ride hours.
- Analyze the effect of weather conditions on rides.
- Understand temperature, humidity, and windspeed patterns.
- Build a visual dashboard for better data insights.

**Dataset**
- The dataset contains information about ride conditions including:
- datetime
- season
- holiday
- workingday
- weather
- temperature
- humidity
- windspeed

**Tools and Technologies**
- Python (Pandas, Matplotlib)
- Data Cleaning and Exploratory Data Analysis (EDA)
- Power BI Dashboard
- GitHub for project documentation

**Key Analysis**
- Peak ride demand by hour
- Weather distribution analysis
- Temperature trends throughout the day
- Humidity vs windspeed relationship

**Project Workflow**
- Data loading and cleaning
- Feature extraction from datetime
- Exploratory data analysis
- Data visualization
- Power BI dashboard creation
- Business insights generation
